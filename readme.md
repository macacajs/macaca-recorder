通过录制的方式生成 E2E 测试的代码，支持插件，降低 E2E 测试的手动书写难度。

## 整体设计

以 Node.js 作为主流程控制器，用 Playwright 实现页面录制，汇总事件流，通过事件流生成对应的代码。

## 目录

```
src/
├── core // 核心机制，提供插件和注入能力
├── injected // 插入测试页面
│   ├── isomorphic
│   └── lib
├── node // node层逻辑，启动逻辑成功程序入口
│   ├── plugins
│   │   ├── api  // 对外报漏api
│   │   ├── assets-server // 本地文件服务
│   │   ├── browser   // playwright浏览器功能
│   │   ├── event     // 事件
│   │   └── generator // 生成器核心逻辑
│   │       ├── generated
│   │       └── page
├── recorder // 代码编辑器页面
│   └── services
└── utils
```
